# theaquarix.github.io

Fast static GitHub Pages site for TheAquarix, an SEO expert portfolio.

## Performance choices

- No framework or build step.
- No third-party scripts.
- System fonts only.
- Optimized responsive hero images with WebP and JPEG fallback.
- Semantic HTML, canonical URL, Open Graph/Twitter metadata, JSON-LD, sitemap, and robots.txt.

## Publish on GitHub Pages

Create a public repository named `theaquarix.github.io` under the `theaquarix` GitHub account, then push these files to the repository's default branch. GitHub Pages will serve it at `https://theaquarix.github.io/`.
